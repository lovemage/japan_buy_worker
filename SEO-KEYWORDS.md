## 核心痛點定位

### 場景

走進日本藥妝店，面對數千種商品，時間有限。想快速找到特價品、退稅商品，幫親友代購賺回機票錢。但傳統流程讓你把寶貴的出國時間全耗在重複性工作上：

1. **研究匯率** — 手動查匯率、逐一換算台幣售價
2. **商品描述** — 看不懂日文，要查翻譯、手打品名規格
3. **翻譯** — 外文商品名稱翻成中文，一個一個查
4. **拍照** — 拍完還要整理、裁切
5. **群組貼文** — 在 LINE 群/IG 逐一發佈商品圖文
6. **回覆詢問** — 重複回答「這個多少錢」「還有嗎」「怎麼算」
7. **整理訂單** — 小幫手或自己從聊天記錄中人工彙整訂單

### 解決方案

**vovosnap = LINE 群 / IG 的代購增強工具。**

不是取代你的 LINE 群或 IG，而是讓它們更強大：
- **一次設定**：匯率 + 利潤，之後不用再管
- **只需拍照**：AI 自動辨識商品名稱、翻譯、計算售價、上架
- **分享連結**：把商店連結丟到 LINE 群 / IG 限動，客人自助瀏覽下單
- **專注兩件事就好**：拍照 + 邀請朋友加入

效率提升至少 10 倍。出國時間用來逛街，不是用來打字。

### 目標用戶動機

- 出國旅遊，順便幫親友代購**賺回機票錢**
- 導遊帶團，想靠代購增加額外收入
- 專職代購者，需要更快的上架速度來服務更多客人

> **SEO 訊息核心：** 所有內容圍繞「出國時間有限 → 拍照就搞定 → 賺回機票錢」。vovosnap 是 LINE 群/IG 的增強工具，不是獨立電商平台。

## 十一、技術 SEO 注意事項

- **URL 結構**：使用 `/s/{slug}/` 子資料夾（已實作），有利 domain authority 集中
- **多租戶 SEO**：每個商店頁面需獨立的 title、meta description、OG tags
- **行動優先**：確保所有頁面 Core Web Vitals 達標（Cloudflare Workers 已有優勢）
- **Sitemap**：動態生成 XML sitemap 包含所有商店和指南頁
- **Robots.txt**：允許搜尋引擎爬取商店頁面和內容頁
- **Canonical URL**：子網域 `{slug}.vovosnap.com` 需設定 canonical 到 `vovosnap.com/s/{slug}/`
