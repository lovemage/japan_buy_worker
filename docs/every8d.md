1.1 取得連線憑證
1.1.1 功能說明
本功能提供取得服務連線資源憑證的功能，建議每 8 小時取一次 token。
1.1.2 網路請求
 網址：https://[SiteUrl]/API21/HTTP/ConnectionHandler.ashx。
 方式：POST。
 內容類型：application/json。
 參數：說明詳見 1.1.3。
1.1.3 參數說明
 請求參數如表 1-1-1，範例詳見 1.1.4。
 回應參數見表 1-1-2，範例詳見 1.1.5。
表 1-1-1 取得連線憑證請求參數
Body
參數名稱 類型 M/O* 說明
HandlerType int M 處理類型；值為 3。
VerifyType int M 驗證類型；值為 1。
UID string M 帳號。
PWD string M 密碼。
* M 為必填；O 為選填。
表 1-1-2 取得連線憑證回應參數
參數名稱 類型 M/O* 說明
Result boolean M
取 得 結 果 ； 值 為 true(成 功)或 false(失
敗)。
Msg string M
結果說明，例：成功回傳 Token 或失敗回
傳「異常訊息」。
* M 為必填；O 為選填。
